CS3210 C++ Primers

This folder contains primers for C++ for those who are not familiar with the language. In CS3210, we will be using C++ for all labs and tutorials, and we recommend it for assignments.

We include both a short and long primer for C++ in this folder.

  • The short primer is a quick overview of some of the important features of C++ that are different from C.

  • The long primer is a more detailed overview of C++ for those who are interested in learning more about the language.

  • Note that there will be significantly overlap between these two - the long primer is a superset of the short primer.

Keep in mind that we don't need you to become a C++ expert (or even be reasonably good at it) in this module. We are simply using it instead of C to give you (optional) access to more powerful features that will make your life easier. Please feel free to only use what you need, and you could still write your C++ code like C if you want to.
